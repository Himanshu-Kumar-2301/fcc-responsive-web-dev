# FreeCodeCamp Responsive Web Design Projects

This repository contains my practice projects and labs from the **Responsive Web Design Certification** on freeCodeCamp.  
It showcases my progress in **HTML, CSS, accessibility, responsive layouts, and animations**.

---

## 📂 Project Categories

### HTML

- **Basic HTML**: Cat Photo App, Recipe Page, Bookstore Page, Travel Agency Page, Music/Video Players, Heart Icon, Video Compilation Page
- **Semantic HTML**: List of Major Web Browsers, Quency's Job Tips Page, Cat Blog Page, Event Hub
- **Forms & Tables**: Hotel Feedback Form, Final Exams Table, Book Catalog Table
- **Survey Form Project**
- **Accessibility**: Tech Conference Schedule Table, Accessible Audio Controller, Checkout Page, Movie Review Page, Multimedia Player

### CSS

- **Basic CSS**: Café Menu, Business Card, To‑Do List, Blog Post Card
- **Units & Positioning**: Event Flyer, Cat Painting, House Painting
- **Pseudo Classes & Elements**: Greeting Card, Parent Teacher Conference Form, Job Application Form
- **Colors**: Colored Markers, Colored Boxes
- **Styling Forms**: Registration Form, Contact Form, Game Settings Panel, Feature Selection Page
- **Box Model**: Rothko Painting, Confidential Email Page
- **Flexbox**: Photo Gallery, Colorful Boxes, Price Plans Layout
- **Typography**: Nutritional Label, Newspaper Article
- **Accessibility**: Quiz Webpage, Tribute Page
- **Selectors**: Balance Sheet
- **Responsive Design**: Piano
- **Grid**: Magazine, Newspaper Layout
- **Projects**: Technical Documentation Page, Product Landing Page, Personal Portfolio
- **Animations**: Ferris Wheel, Moon Orbit, Flappy Penguin
